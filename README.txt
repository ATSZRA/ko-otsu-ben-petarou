甲乙弁ぺた郎 HTML版

使い方:
1. index.html をブラウザで開きます。
2. PDFを選びます。
3. PDF表示枠で1ページ目のプレビューを確認します。
4. 「甲・乙・丙・弁」、番号、右上/左上を指定します。
5. 必要に応じて「ダウンロードフォルダを選択」から保存先フォルダを選びます。
6. 「PDFを作成」を押します。

PDFはブラウザ内で処理され、外部へ送信されません。
スタンプされるのは1ページ目だけです。
生成ファイル名は「甲001.pdf」のように、種別 + 半角3桁番号になります。
保存先フォルダを選ばない場合は、ブラウザ既定のダウンロードフォルダに保存されます。
フォルダ選択は対応ブラウザ（Chrome/Edge等）で利用できます。

同梱ライブラリ:
- pdf-lib 1.17.1
- PDF.js 3.11.174
